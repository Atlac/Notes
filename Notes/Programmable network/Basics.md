# Network-Core Functions

#### Routing:
source-destination route taken by the packet
#### Forwarding:
move packet from input to the correct output

# Internet Protocol Stack
#### Application: 
supporting network applications (FTP, SMTP, HTTP)
#### Transport:
process-process data transfer (TCP, UDP)
#### Network:
routing of datagrams from source to destination (IP, routing protocols)
#### Link:
data transfer between neighboring network elements (Ethernet, 802.111(WiFi), PPP)
#### Physical:
bits "on the wire"

# Encapsulation
Process of concatenating layer-specific headers with a service data unit for transmitting information over computer networks. The order follows: 
application $\rightarrow$ transport $\rightarrow$ network $\rightarrow$ link $\rightarrow$ physical

_Switches_ act on link layer, _Router_  on network layer
